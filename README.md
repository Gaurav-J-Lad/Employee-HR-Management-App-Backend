A full-stack **Employee HR Dashboard Application** built using **Angular (Frontend)** and **Spring Boot (Backend)**.  
It allows HR management operations like employee CRUD, department/position/experience dropdown handling with a clean REST API architecture.

## 📌 API Endpoints

## 🔐 Authentication APIs

| Method | Endpoint           | Description   |
| ------ | ------------------ | ------------- |
| POST   | /api/auth/register | Register User |
| POST   | /api/auth/login    | Login User    |

---

## 👨‍💼 Employee APIs

| Method | Endpoint        | Description        |
| ------ | --------------- | ------------------ |
| POST   | /employees      | Add employee       |
| GET    | /employees      | Get all employees  |
| GET    | /employees/{id} | Get employee by ID |
| PUT    | /employees/{id} | Update employee    |
| DELETE | /employees/{id} | Delete employee    |

---

## 📊 Dropdown / Master Data APIs

| Method | Endpoint               | Description           |
| ------ | ---------------------- | --------------------- |
| GET    | /employees/departments | Get departments       |
| GET    | /employees/positions   | Get positions         |
| GET    | /employees/experiences | Get experience levels |
